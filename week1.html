Week 1: Foundations & OWASP Top 10

Home > Week 1

Day 1: Introduction to Web Technologies

  • HTTP/HTTPS:
    • Read about the basics of the HTTP protocol.
    • Understand the difference between HTTP and HTTPS.
    • Learn about common HTTP request methods (GET, POST, PUT, DELETE).
    • Study HTTP status codes (200, 301, 404, 500).
  • HTML & JavaScript:
    • Take a beginner's course on HTML and JavaScript on Codecademy or a similar platform.
    • Focus on understanding tags, attributes, and basic DOM manipulation.

Day 2: Understanding Databases and SQL

  • SQL Basics:
    • Learn the fundamentals of SQL and relational databases.
    • Practice basic queries: SELECT, INSERT, UPDATE, DELETE, WHERE, ORDER BY.
    • Understand database structure: tables, rows, columns, primary keys.
  • Web Architecture:
    • Study how web applications work: frontend, backend, and database interactions.
    • Learn about web servers, application servers, and database servers.
  • Resources:
    • SQLBolt (interactive SQL tutorial)
    • W3Schools SQL Tutorial

Day 3: Introduction to OWASP Top 10

  • OWASP Top 10 Overview:
    • Read the complete OWASP Top 10 2021 list.
    • Focus on understanding the first 3 vulnerabilities in detail:
      • A01: Broken Access Control
      • A02: Cryptographic Failures
      • A03: Injection
  • Broken Access Control:
    • Learn about authorization vs authentication.
    • Understand vertical and horizontal privilege escalation.
    • Study IDOR (Insecure Direct Object References).
  • Resources:
    • OWASP Top 10 official documentation
    • PortSwigger Web Security Academy

Day 4: Continuing OWASP Top 10

  • Continue OWASP Top 10:
    • Study vulnerabilities 4-7:
      • A04: Insecure Design
      • A05: Security Misconfiguration
      • A06: Vulnerable and Outdated Components
      • A07: Identification and Authentication Failures
  • Security Misconfiguration:
    • Learn about default credentials.
    • Understand directory traversal and information disclosure.
    • Study HTTP security headers (X-Frame-Options, CSP, etc.).
  • Authentication Failures:
    • Learn about session management vulnerabilities.
    • Understand brute force attacks and account enumeration.
  • Practice:
    • Start reading bug bounty reports on HackerOne Hacktivity or similar platforms.

Day 5: Completing OWASP Top 10 & Lab Setup

  • Complete OWASP Top 10:
    • Study the remaining vulnerabilities:
      • A08: Software and Data Integrity Failures
      • A09: Security Logging and Monitoring Failures
      • A10: Server-Side Request Forgery (SSRF)
  • SSRF Focus:
    • Understand what SSRF is and how it works.
    • Learn about different SSRF attack vectors.
    • Study blind SSRF and out-of-band techniques.
  • Lab Setup:
    • Download and install Burp Suite Community Edition.
    • Set up your browser to use Burp Suite as a proxy.
    • Install OWASP Juice Shop using Docker or npm.
  • Initial Practice:
    • Complete the first 2-3 challenges in OWASP Juice Shop.

Day 6-7: Weekend Practice & Review

Day 6: Hands-on Practice

  • OWASP Juice Shop:
    • Complete 5-10 more challenges.
    • Focus on broken authentication, injection, and XSS challenges.
    • Document your solutions and learning points.
  • Burp Suite Familiarization:
    • Practice intercepting requests and responses.
    • Learn to use the Repeater tool.
    • Understand how to modify requests.

Day 7: Review & Plan

  • Week Review:
    • Review all the notes you've taken during the week.
    • Identify areas where you need more practice.
    • Create a summary document of key concepts learned.
  • Preparation for Week 2:
    • Read about XSS types in preparation for next week.
    • Set up additional lab environments if needed.
  • Community Engagement:
    • Join bug bounty Discord servers or Twitter communities.
    • Follow security researchers and read their blogs.
