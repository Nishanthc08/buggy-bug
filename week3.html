Week 3: Tools & Reconnaissance

Home > Week 3

Day 1: Burp Suite Mastery

  • Burp Suite Intruder: Learn the different attack types and use Intruder for fuzzing, rate limit testing, and brute force attacks.
  • Burp Suite Comparer & Decoder: Learn to use Comparer to analyze differences in responses and practice with Decoder for encoding/decoding data.
  • Burp Suite Extender: Browse the BApp Store and install popular extensions like Logger++, Autorize, and Flow.
  • Practice: Re-do previous labs using Burp Suite to automate and speed up the process.

Day 2: Network Reconnaissance with Nmap

  • Nmap Basics: Install nmap and learn basic syntax, host discovery, and different scan types (TCP SYN, TCP connect, UDP).
  • Service Detection & NSE: Use flags for version detection (-sV) and default scripts (-sC). Learn about the Nmap Scripting Engine for vulnerability detection.
  • Practice: Scan vulnerable VMs or Docker containers. Note: Only scan systems you own or have explicit permission to test.

Day 3: Web Directory & File Discovery

  • Directory Brute Forcing: Install and learn tools like Gobuster, Dirb, and ffuf.
  • Wordlist Management: Download and organize SecLists and learn to create custom wordlists.
  • File Extension Discovery: Search for backup files (.bak, .old), config files (.config, .xml), and sensitive files (.env, .git).
  • Practice Commands: 
    gobuster dir -u http://target.com -w /path/to/wordlist
dirb http://target.com
ffuf -w wordlist.txt -u http://target.com/FUZZ

Day 4: Subdomain Enumeration & DNS Reconnaissance

  • Subdomain Discovery: Install and learn tools like Sublist3r, Assetfinder, and Amass.
  • DNS Enumeration: Use dig, nslookup, and DNSrecon for DNS queries and zone transfers.
  • Certificate Transparency & Google Dorking: Use crt.sh and advanced Google search operators to find subdomains and sensitive information.
  • Practice Commands: 
    sublist3r -d target.com
assetfinder target.com
amass enum -d target.com

Day 5: Information Gathering & OSINT

  • Email Harvesting & Tech Stack ID: Use theHarvester for email enumeration and Wappalyzer or Whatweb to identify web technologies.
  • GitHub & Code Repository Recon: Search for exposed API keys and secrets using tools like GitDorker and TruffleHog.
  • Shodan: Create a Shodan account and learn to find exposed services and devices.
  • Practice Commands: 
    theharvester -d target.com -b google
whatweb target.com
shodan search "target.com"

Day 6-7: Weekend Automation & Integration

Day 6: Reconnaissance Automation

  • Bash Scripting: Learn basic bash scripting to combine multiple tools into a workflow.
  • Tool Integration: Learn to pipe outputs between tools and create one-liners for common tasks.
  • Reconnaissance Frameworks: Try ReconNG, OWASP Amass, and Nuclei.

Day 7: Comprehensive Practice

  • End-to-End Recon: Pick a target and perform complete reconnaissance.
  • Week 3 Review: Create a reconnaissance checklist and organize your tools and commands.
  • Prepare for Week 4: Research bug bounty platforms and read about responsible disclosure.
